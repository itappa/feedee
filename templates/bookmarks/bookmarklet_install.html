How to Use

  1. Visit any webpage you want to save
  2. Click the 'Save to Feedee' bookmarklet in your bookmarks bar
  3. Optionally select text before clicking to add it as a description
  4. A notification will confirm the bookmark was saved
  5. Access your bookmark anytime from the Bookmarks page
